## 🚀 Getting Started

### Prerequisites

- Code Editor: Visual Studio Code, Visual Studio, or similar
- Browsers: Modern browser (Chrome, Firefox, Safari, Edge)
- **For ASP.NET Projects**:
  - .NET Framework 4.7.2+ or .NET Core 3.1+
  - SQL Server (LocalDB or full edition)
  - Visual Studio or Visual Studio Code

### Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/IrajFatima/WebPractice.git
   cd WebPractice
   ```

2. **Navigate to a Homework Folder**
   ```bash
   cd homework1
   ```

3. **For HTML/CSS/JavaScript Projects**
   - Open `index.html` in your web browser
   - Or use Live Server extension in VS Code

4. **For ASP.NET Projects**
   ```bash
   # Restore NuGet packages
   dotnet restore
   
   # Build the project
   dotnet build
   
   # Run the project
   dotnet run
   ```
